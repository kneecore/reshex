          Thoth's cousin's Gambit (list of patterns, list → list)

          • Renders
            • Palettes
              Your browser does not support visualizing patterns. Pattern code: adadaqadada

              Works the exact same as Thoth's Gambit however, it does not copy the stack into the sub-stack every iteration. I have not found a better name for this pattern in any texts.


              Garbage Reflection (→ garbage)

              • Renders
                • Palettes
                  Your browser does not support visualizing patterns. Pattern code: awaawawqaqwedadadewwd

                  Simply returns a Garbage iota. It seems to be intended to be used with Garbage Purification.


                  Garbage Purification (garbage → number)

                  • Renders
                    • Palettes
                      Your browser does not support visualizing patterns. Pattern code: eeeaqqwqwqq

                      Takes a Garbage iota, and returns a random number. I find this extremely similar to the Entropy Reflection pattern, the text I have found it from claims this is more "thematically appropiate".
